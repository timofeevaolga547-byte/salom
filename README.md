# SkyWay Travel — Туристический сайт

Учебный проект по веб-разработке. 5-страничный сайт туристического агентства.

## Структура проекта

```
skyway/
├── index.html              # Главная страница
├── css/
│   └── style.css           # Все стили сайта
├── js/
│   └── script.js           # JavaScript: меню, анимации, форма, фильтр
├── pages/
│   ├── about.html          # О компании (+ аудио + видео)
│   ├── destinations.html   # Направления (+ фильтр карточек)
│   ├── tours.html          # Туры (+ таблица сравнения)
│   └── contacts.html       # Контакты (+ форма + карта)
└── README.md               # Этот файл
```

## Выполненные требования

### Организация — 10 баллов
- Сайт организован в папке skyway/
- Изображения через Unsplash CDN
- CSS в папке css/
- JS в папке js/
- HTML/CSS/JS проверены валидатором

### Требования к сайту — 24 балла
- 5 страниц: index, about, destinations, tours, contacts
- Семантические теги: header, nav, main, section, footer
- Комментарии во всех HTML, CSS, JS файлах
- Таблица на странице tours.html
- Форма с 5 полями: text, email, select, tel, textarea
- Адаптивность: media queries для 1024px, 800px, 480px
- Meta viewport на всех страницах
- Оригинальный контент на тему туризма

### Навигация — 16 баллов
- Интуитивная навигация на всех страницах
- Функциональные ссылки между страницами
- Единая навигация на всех страницах
- Внешние ссылки: TripAdvisor, Booking, Skyscanner, WhatsApp, W3C
- Якорные ссылки: #hero, #stats, #features, #useful-links
- Адаптивный контент, вертикальная прокрутка

### Стиль — 20 баллов
- Контраст: тёмный текст на светлом фоне и наоборот
- Шрифты 16-17px, читабельны
- 2 шрифта: Playfair Display (заголовки) + Raleway (текст)
- filter, transform, gradient — в CSS
- Media queries для телефона и ПК
- Стилизованная навигация (таблетки, hover)
- CSS переменные, градиенты, тени, border-radius, анимации

### Медиа — 10 баллов
- Все изображения имеют атрибут alt
- `<video>` элемент на about.html
- `<audio>` элемент на about.html
- Медиа соответствуют теме туризма
- Pixabay CC0 + Unsplash (бесплатные лицензии)

### Интерактивность — 20 баллов
- JS 1: mobileMenu() — переключение мобильного меню
- JS 2: IntersectionObserver — анимация появления элементов
- JS 3: animateCounter() — анимированные счётчики цифр
- JS 4: Фильтрация карточек направлений по категориям
- JS 5: Кнопка «Наверх» с плавной прокруткой
- Обработка данных формы на стороне клиента с валидацией
- Динамически генерируемое сообщение-подтверждение формы
- Анимации CSS + JS (fadeInDown, fadeInUp, pulse)
